# E2E Manual Test Checklist

`npm run dev` 실행 후 아래 항목을 순서대로 확인합니다.

## 1. 인증

- [ ] `/auth/register` - 회원가입 (이메일/비밀번호)
- [ ] `/auth/login` - 로그인
- [ ] 로그인 후 `/projects` 리다이렉트 확인
- [ ] 비로그인 상태에서 `/project/*` 접근 시 로그인 페이지로 리다이렉트
- [ ] 로그아웃 후 세션 만료 확인

## 2. 프로젝트 생성

- [ ] `/new` - 새 프로젝트 생성 폼 표시
- [ ] 제목, 장르, 설명 입력 후 생성
- [ ] `/project/[id]` 로 이동 확인

## 3. Research 단계

- [ ] `/project/[id]/research` - Research 페이지 로드
- [ ] AI 질문 생성 요청
- [ ] 사용자 답변 입력
- [ ] 리서치 결과 표시

## 4. Outline 단계

- [ ] `/project/[id]/outline` - Outline 페이지 로드
- [ ] AI 개요 생성 요청
- [ ] 챕터 드래그 앤 드롭 순서 변경
- [ ] 챕터 추가/삭제/편집

## 5. Write 단계

- [ ] `/project/[id]/write` - Write 페이지 로드
- [ ] 챕터별 AI 집필 요청 (스트리밍)
- [ ] TipTap 에디터에서 텍스트 편집
- [ ] 페이지 에디터 레이아웃 확인 (용지 크기 선택)
- [ ] AI 채팅 (챕터별)

## 6. Edit 단계

- [ ] `/project/[id]/edit` - Edit 페이지 로드
- [ ] AI 편집 분석 요청
- [ ] 편집 히스토리 확인

## 7. Review 단계

- [ ] `/project/[id]/review` - Review 페이지 로드
- [ ] AI 평가 요청 (Pass/Revise)
- [ ] 평가 결과 표시

## 8. Preview & Export

- [ ] `/preview/[id]` - 미리보기 페이지 로드
- [ ] PDF 내보내기
- [ ] EPUB 내보내기

## 9. 메타데이터 & ISBN

- [ ] 메타데이터 폼 열기 (부제목, 저자, 출판사 등)
- [ ] 저자 추가/편집/삭제/순서변경
- [ ] 카테고리 추가 (BISAC/KDC/DDC/custom)
- [ ] 메타데이터 저장
- [ ] ISBN 입력 및 유효성 검사
- [ ] 테스트용 ISBN 생성
- [ ] 바코드 미리보기 및 다운로드

## 10. 표지 디자인

- [ ] 표지 생성 요청
- [ ] CMYK 변환

## 11. 다크/라이트 모드

- [ ] 라이트 모드 전체 페이지 확인 (텍스트 가독성)
- [ ] 다크 모드 전체 페이지 확인 (텍스트 가독성)
- [ ] 모드 전환 시 깜빡임 없음

## 12. 파일 업로드

- [ ] DOCX 파일 업로드 및 파싱
- [ ] PDF 파일 업로드 및 파싱
- [ ] TXT 파일 업로드 및 파싱
